# Verilog History

* Verilog created in the Mid-80s
* Created for simulation
* C like in syntax.
* case-sensitive
* Weakly typed
* Standards:
    * Verilog-95
    * Verilog 2001
        * Supported by most EDA tools
        * Target for this tutorial
    * Verilog 2005
    * SystemVerilog
